# AI-Driven Multi-Agent Phishing & Smishing Detection Framework

This project implements a hybrid AI-driven multi-agent framework
to detect phishing URLs and smishing SMS messages for secure
FinTech communication.

## Agents
- SMS Smishing Detection Agent (ML-based)
- URL Phishing Detection Agent (ML-based)
- FinTech Rule-Based Agent
- Decision Fusion Agent

## How to Run
1. Install Python 3.9+
2. pip install -r requirements.txt
3. python main.py

## Output
The system returns:
- Risk Level (LOW / WARNING / CRITICAL)
- Security Action (ALLOW / VERIFY / BLOCK)
